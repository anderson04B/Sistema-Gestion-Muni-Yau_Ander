Ejecución del Sistema

Sigue estos 3 pasos para correr el programa en tu computadora.

1. Requisitos Previos

Asegúrate de tener Python instalado. Las librerías necesarias están en requirements.txt.

2. Instalación

Abre tu terminal en la carpeta del proyecto y ejecuta:

pip install -r requirements.txt


3. Ejecutar Aplicación

Para iniciar el sistema, usa el siguiente comando:

streamlit run visual_app.py
